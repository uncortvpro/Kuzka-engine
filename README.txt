Папки для разработчика:
-models
-view
-controllers
-config

При первом запуске:
-Заходите в config->db.php и выставляете свои значения базы данных

!Шаблон для всех страниц по умолчанию лежит в папке layout->default.php, для изменения layout: $this->view->layout="Нзвание_файла";

Для создания новых маршрутов:
-Заходите в config->routes.php и выставляете свои значения

Для создание контроллеров:
-Заходите в controllers и создаете контроллер, !!!Первая буква должна быть большой и название 
должно заканчиваться Controller!!! Пример: AccountController
-В начале файла вам необходимо создать namespace application\controllers; и use application\core\Controller;
-Название класса должно быть такое же самое как и название контроллера, и наследовать
абстрактный класс Controller

Для создание вида:
-Заходите во view, создаете папку и файл такое же как controller и action. 
Например: если controller=account, action=register, то необходимо создать папку account, а в нем файл register

Для создания модели:
-Заходите в models, создаете модель, название модели должно быть такое же самое как и название контроллера
и должно начинаться с большой буквы
-В начале файла вам необходимо создать namespace application\models; и use application\core\Model;
-Название класса должно быть такое же самое как и название модели, и наследовать
абстрактный класс Model

Для передачи данных во view:
-В конструкторе необходимо написать массив, название массива всегда должно быть $data
Пример:  $data=['name'=>'Sergey','age'=>21]
Передача: $this->view->render('Главная страница',$data); - в ковычках прописуете title для каждой страницы
Вывод во view: <p>Имя: <?=$name?></p>  <p>Возвраст: <?=$age?></p>

Для передачи данных из Модели в Контроллер:
Используете метод $this->db->row("SQL Запрос")
Пример: 
Model:	$result=$this->db->row("SELECT `title`,`desc` FROM `news`");
        return $result;
Constructor:  $result = $this->model->Название_Метода_Модели;  

Для редиректа:
Используйте метод $this->view->location('ПУТЬ'); или $this->view->redirect('ПУТЬ')
